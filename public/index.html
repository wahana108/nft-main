Welcome and Rules

Comment Policy

- Gunakan bahasa yang sopan dan mendukung komunitas.
- Tidak ada spam, ujaran kebencian, atau promosi tidak relevan.
- Hormati pendapat lain—diskusi untuk kemajuan bersama.

Aturan Dasar

- Semua NFT dibuat untuk tujuan charity—pastikan deskripsi jelas.
- Harga default: Rp100,000 (untuk saat ini).
- Login hanya untuk user terdaftar—kontak admin untuk akses.

Konsekuensi Anomali

- Aktivitas anomali (penipuan, spam, manipulasi buyback) akan dipantau.
- Konsekuensi: Akun dinonaktifkan permanen oleh admin.
- If system failure - Wild Dog Protocol activated.
